Sudoku Solver

I created this application in June of 2020 during a break between classes at Brandeis.

When launched, it provides a simple UI that allows users to enter the digits they're given in a Sudoku puzzle.

The application uses a brute force algorithm to test possible combinations until a solution is found. In also includes some unit tests using JUnit.
